# fullstack-js-mongo-react

run both npm dev-server and dev-bundle scripts (in 2 separate terminals):

$ npm run dev-bundle
$ npm run dev-server

Then open up your browser on ```http://localhost:4040/``` you should see the React application rendered. This application should also be rendered if you disable JavaScript in your browser!

  webpack / babel server - npm run dev
  express server - npm start
